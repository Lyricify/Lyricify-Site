## Lyricify 4

**Lyricify 4** 是一款专为 Spotify 设计的歌词软件，提供滚动歌词和更多功能，全名为 **Lyricify for Spotify**，
是 **Lyricify** 系列作品的一大里程碑

**Lyricify** 可以为您提供细腻、流畅的动画和海量歌词，提升您的 Spotify 音乐体验

### Lyricify 4 Original

Lyricify 4 是目前 Lyricify 系列中最具创新的功能结晶，在这里列出部分首创于 Lyricify 的功能

- **灵动词岛**
    - 首创于 2022.09.30
    - 公开于 `Lyricify 3.8.1-release`
- **妙控条**
    - 首创于 2022.10.09
    - 公开于 `Lyricify 4.0.0.230121-release`
- **歌词界面使用动态专辑封面**
    - 首创于 2022.11.14
    - 公开于 `Lyricify 4.0.0.230121-release`
- **Lyricify Syllable**
    - 首创于 2023.03.13
    - 公开于 `Lyricify 4.0.8.230315-release`
- **Lyricify 智能引擎**
    - 首创于 2023.04.26
    - 公开于 `Lyricify 4.1.0.230503-release`

## Lyricify Open Source

Lyricify 的歌词处理相关代码 均已在
[GitHub 上的 **WXRIW/Lyricify-Lyrics-Helper** ](https://github.com/WXRIW/Lyricify-Lyrics-Helper)
中使用 **Apache License 2.0** 协议开源，希望能为开源社区做出贡献

## Lyricify Mobile

**Lyricify Mobile** 为 Lyricify 家族的移动平台版本，目前支持以下平台：
- Android
- iOS
- iPadOS
- visionOS
- Windows
- macOS

:::note[另请参阅]
若要了解更多有关使用 **Lyricify Mobile** 的信息，请参阅
[GitHub 上的 **Lyricify Mobile 使用指南**](https://github.com/WXRIW/Lyricify-App/blob/main/docs/Lyricify%20Mobile/README.md)
:::
:::tip
若在 Lyricify Mobile 中遇到歌词不正确的情况，您可以在 Lyricify 4 的**歌词管理**中进行修改
:::
