Lyricify 自发布以来受到普遍好评，并推出了不同版本和更新，不断提升用户体验。

当前最新版本是 **Lyricify 4** (Lyricify for Spotify)，包含最新的功能更新和支持，建议所有用户使用。

## Lyricify 4

![Lyricify 4 界面展示](@assets/get-started/versions/lyricify-4-zh-cn.png)

Lyricify 4 是 Lyricify 系列的最新产品。整个歌词处理和动画逻辑进行回炉重造，打造出了迄今为止最完美的 Spotify 歌词软件。

### 特色功能

- 精确到音节的逐字歌词<sup>1</sup>
- 支持使用来自 QQ音乐、网易云音乐、酷狗音乐、MusixMatch、Apple Music 等平台的歌词<sup>2</sup>
  - 还可与 [Lyricify Mobile](#lyricify-mobile) 共用歌词库
- 1:1 还原的 Apple Music 样式歌词
- 完全的触摸屏支持
- 先进的管理工具和高度自定义选项
- Spotify 自定义 API 支持

### 支持平台

- **Windows** (64位, 32位, arm)
  - 要求 Windows 7 SP2 或更新版本
- **macOS** (Intel)
  - 要求 macOS Mountain Lion 或更新版本，最高支持 macOS Mojave
- **Linux 发行版** (64位, 32位, arm)
  - 要求 Ubuntu 20.04/22.04/23.04/23.10 或 Debian Bullseye/Bookworm 或 Fedora 38/39

:::tip[Wine 兼容层]
适用于 **Linux 发行版** 和 **macOS** 的 Lyricify 4 通过 [Wine](https://www.winehq.org/) 兼容层运行。

Wine 还支持更多 Linux 版本，请访问 [WineHQ](https://www.winehq.org/) 来获得更多帮助。
:::

### 运行要求

- [.NET Desktop Runtime 6.0](https://dotnet.microsoft.com/zh-cn/download/dotnet/6.0) 运行时
- 至少双核处理器

## Lyricify Mobile

Lyricify Mobile 是针对移动平台设计的 Lyricify 版本，可以为您提供流畅舒适的歌词体验。

Lyricify 4 中内置的移动平台视图也称为 _Lyricify Mobile_。

### 特色功能

可在移动平台上提供：

- 细腻丝滑的滚动歌词
- 与 [Lyricify 4](#lyricify-4) 共用歌词库
  - 支持使用来自 QQ音乐、网易云音乐、酷狗音乐、MusixMatch、Apple Music 等平台的歌词<sup>2</sup>
- 曲目切换和歌词控制
- Spotify 自定义 API 支持

可在 Lyricify 4 中提供：

- 精确到音节的逐字歌词<sup>1</sup>
- 支持使用来自 QQ音乐、网易云音乐、酷狗音乐、MusixMatch、Apple Music 等平台的歌词<sup>2</sup>
- 1:1 还原的 Apple Music 样式歌词
- 完全的触摸屏支持
- Spotify 自定义 API 支持

### 支持平台

- **iOS**
  - 要求 iOS 8.0 或更新版本
- **安卓 (arm)**
  - 要求 Android 5.0 或更新版本
- **Windows**, **macOS** 和 **Linux 发行版**
  - 内置于 [Lyricify 4](#lyricify-4) 中，系统要求参见 [Lyricify 4 支持平台](#支持平台)
