## Lyricify 4

**Lyricify 4** is a fantastic app providing auto-scrolling lyrics and more functions for [Spotify](https://spotify.com), also known as **Lyricify for Spotify**. It is a great milestone of the **Lyricify** series.

**Lyricify** enhances your Spotify music experience by providing detailed, smooth animations and massive lyrics

### Lyricify 4 Original

Lyricify 4 is currently the most innovative product in the Lyricify series. Here are some of the features first launched in Lyricify:

-   **Dynamic Lyrics Island**
    -   First launched on 2022.09.30
    -   Released @ `Lyricify 3.8.1-release`
-   **Magic Stripe**
    -   First launched on 2022.10.09
    -   Released @ `Lyricify 4.0.0.230121-release`
-   **Live Album Cover**
    -   First launched on 2022.11.14
    -   Released @ `Lyricify 4.0.0.230121-release`
-   **Lyricify Syllable**
    -   First launched on 2023.03.13
    -   Released @ `Lyricify 4.0.8.230315-release`
-   **Lyricify Intelligent Engine**
    -   First launched on 2023.04.26
    -   Released @ `Lyricify 4.1.0.230503-release`

## Lyricify Open Source

Lyrics processing code used in Lyricify is open source on [**WXRIW/Lyricify-Lyrics-Helper** on GitHub](https://github.com/WXRIW/Lyricify-Lyrics-Helper) with **Apache License 2.0** agreement, hoping to contribute to the open source community.

## Lyricify Mobile

**Lyricify Mobile** is the mobile version of Lyricify series, runs on:

-   Android
-   iOS
-   iPadOS
-   visionOS
-   Windows
-   macOS

:::note[Also see] To know more about **Lyricify Mobile**, see [**Lyricify Mobile User Guide** on GitHub](https://github.com/WXRIW/Lyricify-App/blob/main/docs/Lyricify%20Mobile/README.md) ::: :::tip If you encounter incorrect lyrics in Lyricify Mobile, you can modify them in **Lyrics Management** in Lyricify 4 :::
